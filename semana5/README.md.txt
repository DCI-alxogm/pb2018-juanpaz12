En el primer (funciones) programa realizamos el cálculo de algunas funciones como logaritmo, seno, coseno, entre otras; el programa lo que hacia era solicitar el rango que habrá en cada función así cono cada cuanto se va realizar (espaciado).
En el segundo programa realizamos el cálculo factorial de un número, para eso se utilizó la función “for”, que lo que hacía era multiplicar aumentando de uno en uno hasta llegar el número determinado, en ese punto la función “for” de detiene, y posteriormente imprime el resultado a la pantalla.
El tercer programa lo que hace es calcular si entre un intervalo dado hay números primos, para eso lo que se hizo fue que dentro la función “for” se sacaba una raíz de uno en uno al numero dado desde el el termino mas pequeño hasta el mas grande, si al dividir el número dado por el usuario entre los términos que hay desde el 2 hasta el su raíz daba un entero, este no era primo, si daba in decimal se trataba de un primo, si era un numero primo se imprimía en la pantalla, de no serlo se descartaba.
El último programa se trataba de convertir de números reales a binario y viceversa, para esto primero se utilizó la función switch, en la cual se preguntaba que se iba a realizar, después de esto solo se realizaban unos pequeños cálculos utilizando la función “for”.
